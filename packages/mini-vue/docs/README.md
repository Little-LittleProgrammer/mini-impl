# mini-vue 文档导航

`packages/mini-vue/docs` 现在按“先建立全局认知，再按主题深入”的方式组织。

## 文档分组

- `01-架构`：项目分层、依赖方向、整体闭环
- `02-运行时`：`VNode`、渲染器、组件系统、调度器、DOM 适配
- `03-响应式`：`reactive`、`ref`、`computed`、`watch`
- `04-编译时`：`parse`、`transform`、`codegen`
- `05-对比`：和完整 Vue、Vue 3.6 的差距与边界

## 推荐阅读顺序

如果你第一次看这个项目，建议按下面顺序进入：

1. [项目总架构](./01-架构/总览.md)
2. [运行时总览](./02-运行时/总览.md)
3. [响应式总览](./03-响应式/总览.md)
4. [编译时总览](./04-编译时/总览.md)
5. [对比总览](./05-对比/总览.md)

如果你已经熟悉 Vue 3，只想快速定位源码，可以直接从各目录下的 `总览.md` 开始，它们会优先给出阅读路径和核心源码入口。
