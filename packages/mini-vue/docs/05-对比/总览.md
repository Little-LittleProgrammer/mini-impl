# 对比总览

这一组文档不解释“怎么实现”，而是解释“当前实现没覆盖什么”以及“它和官方 Vue 差在哪”。

## 推荐阅读顺序

建议按下面顺序阅读：

1. [与完整Vue对比](./与完整Vue对比/总览.md)
2. [与Vue3.6对比](./与Vue3.6对比.md)
3. [Vue3.6响应式升级](./Vue3.6响应式升级.md)

## 适合什么时候看

- 想先理解这份 mini-vue 的能力边界
- 想知道它和真实 Vue 到底差在哪
- 想把“教学实现”和“生产框架”明确区分开

## 建议放在什么时候读

- 第一次读项目时，先看架构、运行时、响应式、编译时，再回来看这里
- 如果你正在补功能，或者想确认某个能力为什么不存在，可以直接先看这里
