## 这套文档解决什么问题

- 帮你先看清 `mini-vue` 的整体分层
- 再拆开运行时、响应式、编译器三条主链路
- 最后明确这份实现与官方 Vue 的能力差距

## 目录分组

- `01-架构`：项目整体结构、依赖方向、模块接缝
- `02-运行时`：`VNode`、渲染器、组件系统、调度器、DOM 适配层
- `03-响应式`：`reactive / ref / computed / watch`
- `04-编译时`：`parse / transform / codegen`
- `05-对比`：与完整 Vue、Vue 3.6 的差异
