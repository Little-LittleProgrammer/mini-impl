# mini-vite 文档导航

`packages/mini-vite/docs` 现在按“先看整体，再按核心模块深入”的方式组织。

## 文档分组

- `总览`：快速入口、请求链路、阅读顺序
- `架构`：启动流程、核心分层、模块协作关系
- `热更新`：HMR 接入方式、模块图、更新边界与客户端运行时
- `插件`：插件接口、插件容器、核心插件与执行流程
- `静态资源`：CSS 处理、静态资源请求、缓存与安全策略
- `预构建`：依赖扫描、esbuild 预构建、缓存与路径重写

## 推荐阅读顺序

如果你第一次看这个项目，建议按下面顺序进入：
